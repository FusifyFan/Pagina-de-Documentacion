Introducción

JavaScript es un lenguaje de programación multiplataforma orientado a objetos. Dentro de un entorno de host (por ejemplo, un navegador web), JavaScript se puede conectar a los objetos de su entorno para proporcionar control sobre ellos.

JavaScript contiene una biblioteca estándar de objetos, como Array, y un conjunto central de elementos del lenguaje, como operadores, estructuras de control y declaraciones. JavaScript se puede ampliar para diversos fines complementándolo con objetos adicionales; por ejemplo:

  • JavaScript del lado del cliente amplía el lenguaje principal al proporcionar objetos para controlar el navegador y su DOM. Por ejemplo, las extensiones del lado del cliente permiten que una aplicación coloque elementos en un formulario HTML y responda a eventos del usuario, como clics del mouse.
  • JavaScript del lado del servidor amplía el lenguaje principal al proporcionar objetos relevantes para ejecutar JavaScript en un servidor. Por ejemplo, las extensiones del lado del servidor permiten que una aplicación se comunique con una base de datos o realicen manipulaciones de archivos en un servidor.

JavaScript y Java

Java y JavaScript son lenguajes de programación muy distintos a pesar de su nombre similar. Java es un lenguaje de propósito general, y se utiliza principalmente para aplicaciones empresariales, móviles (como Android) y de servidor. Su código se compila, y se debe ejecutar en la Java Virtual Machine (JVM), lo que permite que sea multiplataforma. Además, Java es fuertemente tipado, lo que significa que se debe declarar el tipo de dato de las variables, y está diseñado principalmente para la programación orientada a objetos, usando clases y herencia.

Por otro lado, JavaScript es un lenguaje interpretado, inicialmente desarrollado para la web, que se ejecuta en navegadores o en entornos como Node.js. Es de tipado dinámico, lo que significa que las variables pueden cambiar de tipo durante la ejecución. En lugar de basarse en clases como Java, JavaScript usa un modelo basado en prototipos para su orientación a objetos.

Java es más adecuado para aplicaciones de gran escala y sistemas complejos, mientras que JavaScript es popular para la creación de interfaces web interactivas, manipulando el DOM y gestionando eventos en el navegador.

Variables

En JavaScript, las variables son contenedores que almacenan valores y se pueden declarar usando tres palabras clave principales: let y const. Cada una tiene características diferentes en cuanto a su alcance y mutabilidad. A continuación, algunos ejemplos de cómo declararlas:

let y = 5;
let texto = "Hola";
const pi = 3.141592;

Las variables declaradas con let y const están limitadas al bloque donde se declaran (todo lo que está dentro de { }). Esto incluye funciones, bucles y condiciones. Es importante recordar que las variables constantes no pueden ser redefinidas.

Declarando variables - restricciones

En JavaScript, los nombres de las variables deben seguir ciertas reglas y restricciones para ser válidos: debe comenzar con una letra, guion bajo (_) o signo de dólar ($); no puede usar palabras reservadas (if, else, while, for, function, return, class, etc); son sensibles a mayúsculas y minúsculas; no puede contener espacios.

let %var; // Error: el símbolo % no es permitido
let let = 5; // Error: "let" es una palabra reservada
let nombre completo; // Error: los espacios no están permitidos
let nombre# = 'Carlos'; // Error: el símbolo # no está permitido

Siguiendo estas restricciones y buenas prácticas, se puede lograr un código más claro, legible y menos propenso a errores.
